플랫폼(Platform)이란?
단순 정의 : 단상, 무대
단상, 무대 따위의 의미가 바뀌어 컴퓨터 시스템 기반이 되는 하드웨어, 소프트웨어, 응용 프로그램이 실행 될 수 있는 기초를 이루는 컴퓨터 시스템을 의미
공간정보 오픈플랫폼(Spatial Information Open Platform)
공간정보(Spatial Information)
우리가 사는 실세계의 형상과 그것을 바탕으로 도형으로 구성한 물리적인 공간 구성요소(건물, 도로 등)와 논리적인 공간 구성요소(행정경계, 지적 등) 그리고
그 도형에 속한 속성을 모두 포괄하여 공간정보라고 합니다.
공간정보는 표현의 수준에 따라 2차원 공간정보와 3차원 공간정보로 나누어질 수 있습니다.
오픈(Open)
공간정보를 공개합니다.
단순히 볼 수 있게만 하는 것이 아니라, 2차, 3차 활용할 수 있도록 다양한 서비스 체계로 공간정보를 공개합니다.
서비스 체계로 공간정보를 공개합니다.
공간정보 오픈플랫폼은 국가가 보유하고 있는 공개 가능한 공간정보를 모든 국민이 자유롭게 활용할 수 있도록 다양한 방법을 제공합니다.
무엇을 주는가?(What)
국가공간정보
국가가 보유하고 있는 공개 가능한 모든 공간정보를 점진적으로 국민에게 제공합니다.

제공하는 오픈API 종류
지도오픈API 오픈플랫폼이 제공하는 클라이언트 프로그램을 활용하여 새로운 서비스를 보다 쉽게 구축 가능
(오픈플랫폼 클라이언트 + 사용자 보유 콘텐츠 + 기능 추가)
단점 : 오픈플랫폼 클라이언트의 기능적 한계를 벗어나는 서비스를 구축할 수 없음
데이터 오픈API (2013년) 데이터에 대한 직접적인 접근을 통해 보다 다양한 형태의 응용프로그램과 서비스 개발 가능
(오픈플랫폼의 서버 자원까지만 활용하여 데이터에 접근) 단점 : 사용자가 새로운 클라이언트 프로그램을 제작해야 함

얻을 수 있는 효과는?(Expectation)
국가공간정보의 민간 활용
국가공간정보의 본격적인 민간 활용이 본 오픈플랫폼의 궁극적인 목적이며, 이를 통해 민간의 다양한 응용 아이디어가 꽃을 피우기를 기대합니다.
국가공간정보 품질의 선순환
국가공간정보의 활용성과 가치가 높아짐에 따라 품질에 대한 요구도 뒤따를 것이며, 이를 통해 국가공간정보의 품질이 향상되고 다시 활용되는 건강한 공간정보의 라이프 사이클이 만들어질 것입니다.
사용자의 예산규모와 서비스 유형에 맞는 서비스 채널 선택 가능
